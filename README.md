# 🎓 University & Course Data Scraping Assignment

## 📌 Internship Assignment Submission  
**Position:** AI/ML & Web Scraping Data Entry Intern  

This project demonstrates automated web data structuring using Python.

---

## 📊 Objective

The goal of this assignment is to:

- Collect university data
- Collect course data
- Maintain relational integrity
- Structure the data professionally
- Export clean Excel output

---

## 🛠️ Tools Used

- Python
- Pandas
- BeautifulSoup
- Requests
- OpenPyXL

---

## 🏫 Data Collected

### Universities Information

- University Name
- Country
- City
- Official Website

### Courses Information

- Course Name
- Level
- Discipline
- Duration
- Fees
- Eligibility Criteria

Each course is correctly linked to its university using a unique `university_id`.

---

## 🔗 Data Structure

### Sheet 1: Universities

| university_id | university_name | country | city | website |

---

### Sheet 2: Courses

| course_id | university_id | course_name | level | discipline | duration | fees | eligibility |
